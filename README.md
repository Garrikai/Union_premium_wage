# Union_wage
Labour economics project

## Abstract
This study seeks to present empirical evidence of the union wage gap between workers who are either union
members or non-members in the USA labour market. The paper uses cross-sectional data from the Current
Population Survey. We estimate numerous regression specifications, including varying the control variables
on individual household data. Our findings show that there is a significant union wage effect in the sample
included at approximately 17%, and the causal effect between union membership and returns to schooling results
in a reduction in wage inequality. The significance of this study is to provide policy-makers with information
on the importance of union wage effects, especially in developing countries.

## Information
The study employed linear regression and quantile regression. The code for this documentation is embedded in the .Rmd file. This project was compiled in Rmarkdown when l was still a novice in sourcing code and knitting documents.
